# 🎬 Collaborative Recommendation System

A **Collaborative Recommendation System** built using **Python**, **Pandas**, and **NumPy** that recommends movies based on user rating patterns. The project implements **User-Based Collaborative Filtering**, where users with similar preferences are identified to generate personalized movie recommendations.

---

## 📌 Project Overview

Collaborative Filtering is one of the most widely used recommendation techniques in streaming platforms and e-commerce applications. Instead of analyzing movie content, this system recommends movies by comparing users' ratings and identifying users with similar interests.

---

## 🚀 Features

- User-Based Collaborative Filtering
- Personalized Movie Recommendations
- User Rating Analysis
- Movie Rating Matrix
- Data Preprocessing
- Easy-to-understand Python Implementation

---

## 🛠️ Technologies Used

- Python

- NumPy

## 📂 Dataset

This project uses the **MovieLens 100K Dataset**.

Dataset Files:

- `u.data` – User ratings
- `Movie_Id_Titles.txt` – Movie ID and title mapping

## 🔍 Workflow

1. Load the MovieLens dataset.
2. Clean and preprocess the data.
3. Create a User–Movie Rating Matrix.
4. Find similar users based on ratings.
5. Recommend movies liked by similar users.

---

## 📈 Output

The system recommends movies by identifying users with similar interests and suggesting highly rated movies they have enjoyed.
